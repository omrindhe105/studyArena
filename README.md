# StudyArena - Study Management Dashboard

StudyArena is a minimal, distraction-free dashboard that helps you track your study time, productivity, and revision. It's designed to be simple and easy to use, so you can focus on what matters most - learning.

## Features

- Track your study time: StudyArena lets you set timers for your study sessions and provides detailed reports on how much time you spend studying.
- Set goals: Set goals for yourself and track your progress over time. StudyArena provides a goal tracker that helps you stay motivated.
- Track your productivity: StudyArena lets you track your productivity by providing a daily and weekly productivity report.
- Revision tracker: StudyArena lets you track your revision by providing a revision report.

## Tech Stack

- Next.js: A React framework for building server-side rendered and static websites.
- Tailwind CSS: A utility-first CSS framework for rapidly building custom user interfaces.
- Mongoose: An Object Data Modeling (ODM) library for MongoDB and Node.js.
- Lucide React Icons: A set of high-quality SVG icons for React.

## How to run this project

1. Clone this repository.
2. Install the dependencies by running `npm install`.
3. Start the development server by running `npm run dev`.
4. Open your browser and go to `http://localhost:3000`.

## Scripts

- `npm run dev`: Starts the Next.js development server.
- `npm run build`: Builds the Next.js production-ready static files.
- `npm run start`: Starts the Next.js production server.
- `npm run lint`: Runs the ESLint linter on your code.
